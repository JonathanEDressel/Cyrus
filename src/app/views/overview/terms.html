Description of Service

Cyrus is a desktop application for monitoring and automating cryptocurrency trading across multiple exchanges. It runs entirely on your local machine and communicates only with the exchange APIs you configure. Key capabilities include:

  • Connecting to exchanges via user-provided API keys
  • Viewing open orders and whitelisted withdrawal addresses
  • Creating automation rules triggered by order fills or balance thresholds
  • Monitoring live market data and crypto price charts
